# Adoption Gate

Status: `required_before_1_0`

Graphite cannot be used foundation-wide until all gates pass.

## Technical Gates

- [ ] stable protocol object schema,
- [ ] signed-governance verification,
- [ ] docs-path security,
- [ ] claim status model,
- [ ] lab promotion model,
- [ ] integration tests,
- [ ] pentest suite,
- [ ] deployment runbook,
- [ ] migration policy.

## Social Gates

- [ ] at least one external developer reviews the protocol,
- [ ] at least one external developer can run it locally,
- [ ] at least one signed foundation decision approves adoption,
- [ ] maintainership policy exists.

## Foundation Gate

The foundation must sign a governance decision:

```text
adopt-graphite-1-0
```

before Graphite becomes formal infrastructure.
