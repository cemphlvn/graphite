# Non-Adoption Boundary

Status: `active`

Graphite is not yet the official name of the foundation server architecture.

Use:

```text
Graphite incubator
Graphite candidate protocol
Graphite pre-1.0 module
```

Do not use:

```text
foundation runs on Graphite
Graphite is adopted
Graphite protocol is stable
Graphite-certified
```

Reason:

Naming a system too early creates hidden dependency and future confusion.
